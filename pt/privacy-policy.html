Informação que podemos processar

  • Dados introduzidos para gerar ou ler códigos QR.
  • Ficheiros, contactos, eventos, localizações, mensagens ou textos escolhidos pelo utilizador.
  • Informação técnica necessária para lojas, partilha ou funções do dispositivo.

Uso da informação

A informação é usada apenas para executar a função solicitada na aplicação.

Serviços de terceiros

Alguns links podem abrir serviços externos como lojas de aplicações, mapas, email ou navegador.

Armazenamento

Não é criada uma base de dados remota própria para armazenar informação pessoal destas aplicações.

Contacto

Para questões de privacidade, use os canais publicados em juanromerodominguez.dev.
