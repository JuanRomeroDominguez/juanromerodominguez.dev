Privatlivspolitik

Sidst opdateret: 2026-05-01

Juan Romero Domínguez respekterer dit privatliv. Denne politik forklarer, hvilke oplysninger der kan behandles, når du bruger apps, eebsites, digitale værktøjer og relaterede tjenester.

Denne politik er fælles for de apps og tjenester, der er angivet nedenfor. Ikke alle apps bruger alle funktioner, tilladelser, SDKer eller tjenester, der beskrives her.

Oplysninger, vi kan behandle

  • QR-indhold og brugerindtastede oplysninger kan behandles lokalt for at levere den ønskede funktion.
  • Kamera-, billed-, PDF-, lyd- og lokale præferencedata kan behandles, når den relevante app har brug for dem.
  • Firebase Analytics, Firebase Crashlytics, Google AdMob og Google User Messaging Platform kan behandle tekniske data, brugsdata, samtykke- og annonceringsdata efter deres egne politikker.

Formål med behandlingen

  • Vi behandler data for at levere funktionalitet, åbne kompatible apps, gemme eller dele indhold, vise annoncer, administrere samtykke, måle ydeevne, forebygge svindel og forbedre kvaliteten.

Lokal behandling og brugerens kontrol

Hvor det er muligt, behandles QR-indhold og forberedte handlinger lokalt på enheden. Brugeren beslutter, om der skal gemmes, deles, ringes, sendes, åbnes, forbindes, navigeres eller bekræftes.

Omfattede eebsites

Denne politik kan også gælde for eebsites, der drives af Juan Romero Domínguez, herunder calendario.gratis og juanromerodominguez.dev.

Tredjepartstjenester

Vores apps kan bruge Google AdMob, Firebase Analytics, Firebase Crashlytics og Google User Messaging Platform.

Internationale overførsler

Nogle tekniske, annoncerings-, analyse- og samtykkedata kan behandles uden for dit land eller område.

Opbevaring

Oplysninger, der gemmes lokalt, forbliver på enheden, indtil brugeren sletter dem, rydder appdata eller afinstallerer appen.

Børn

Vores apps er beregnet til et generelt publikum, medmindre en konkret butiksfortegnelse angiver andet.

Dine rettigheder og valg

Du kan afinstallere appen, begrænse annonceringsidentifikatorer, administrere samtykke og udøve databeskyttelsesrettigheder ved at kontakte os.

Sikkerhed

Vi bruger rimelige tekniske og organisatoriske foranstaltninger, men intet system er helt sikkert.

Ændringer

Vi kan opdatere denne politik, når vores apps, tjenester, SDKer, juridiske krav eller butikskrav ændres.
