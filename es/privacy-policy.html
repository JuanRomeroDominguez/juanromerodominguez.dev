Información que podemos procesar

  • Datos introducidos por el usuario para generar o leer códigos QR.
  • Archivos, contactos, eventos, ubicaciones, mensajes o textos seleccionados por el usuario.
  • Información técnica necesaria para abrir tiendas, compartir contenido o usar funciones del dispositivo.

Uso de la información

La información se usa únicamente para ejecutar la función solicitada dentro de la aplicación.

Servicios de terceros

Algunos enlaces pueden abrir servicios externos como tiendas de aplicaciones, mapas, correo o navegador. Esos servicios aplican sus propias políticas.

Almacenamiento

No se crea una base de datos remota propia para almacenar información personal introducida en estas aplicaciones.

Contacto

Para consultas de privacidad, usa los canales publicados en juanromerodominguez.dev.
