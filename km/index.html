бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҖбҹ’бһҡбһЎбһ¶бһ…бһҸбҹ’бһҡбһ„бҹ’бһӮ ALV, OOP бһ“бһ·бһ„бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бҹ” - бһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһңбһҸбҹ’бһҗбһ» бһ“бһ·бһ„бһҸбһҪбһ“бһ¶бһ‘бһёбҹ” - BRFplus (Business Rule Framework plus)бҹ” - бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҖбһ¶бһҡбһ„бһ¶бһҡбһ–бһёбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҸбһ¶бһҳбһҡбһҷбҹҲбһҡбһ”бһ¶бһҷбһҖбһ¶бһҡбһҺбҹҚбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ”бһҫбһҖбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ бһ“бһ·бһ„бһҡбһ”бһ¶бһҷбһҖбһ¶бһҡбһҺбҹҚбһҖбһ»бһҳбһ¶бһҡбҹ” - бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһ“бһ·бһ„бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҖбҹҶбһҺбһҸбҹӢбһ бҹҒбһҸбһ»; бһҖбһ¶бһҡбһ”бһ“бҹ’бһҗбҹӮбһҳбһҹбһ¶бһҡ бһҖбһ¶бһҡбһӣбһ»бһ” бһҖбһ¶бһҡбһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбҹ” - бһўбҹҠбһёбһҳбҹӮбһӣбһҒбһ¶бһ„бһҖбҹ’бһҡбҹ…бһ“бһ·бһ„бһҒбһ¶бһ„бһҖбҹ’бһ“бһ»бһ„бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ SAP бҹ” - Emigall (SAP IS-U Migration Workbench)бҹ” - F4 бһҮбһҪбһҷ бһ“бһ·бһ„бһ…бҹҒбһүбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҮбҹҶбһ“бһҪбһҷбһҖбһ¶бһҡбһҹбҹ’бһңбҹӮбһ„бһҡбһҖбҹ” - бһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҹбҹҶбһҺбһҫ (ChaRM) бҹ” - бһҖбһ¶бһҡбһҖбһҸбҹӢбһҸбҹ’бһҡбһ¶бһ”бһүбҹ’бһ…бһјбһӣбһҮбһ¶бһ”бһ¶бһ…бҹӢ (SHDB) бҹ” - бһҖбҹ’бһҡбһ»бһҳ / бһҳбҹүбһјбһҢбһ»бһӣбһҳбһ»бһҒбһ„бһ¶бһҡ RFC бҹ” - SAP InfoSets бһ“бһ·бһ„бһҹбҹҶбһҺбһҪбһҡбҹ” - бһҳбҹүбһәбһ“бһ»бһҷбһҸбҹҶбһ”бһ“бҹӢбҹ” - бһўбһ¶бһ„бһҳбҹүбһјбһҢбһ»бһӣбҹ” - бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҸбһ¶бһҳбһҠбһ¶бһ“ бһ“бһ·бһ„бһ–бҹҒбһӣбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбҹ” - бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҹбҹ’бһҡбһ”бһӮбҹ’бһ“бһ¶бҹ” - бһўбһ¶бһ„бһ…бһ·бһүбҹ’бһ…бһ№бһҳбһҸбҹ’бһҡбһёбҹ” - Psobwork (PSCD: Contract Object) бҹ” - бһ…бҹҶбһҺбһ»бһ…бһ”бҹ’бһҡбһҹбһҫбһҡбһЎбһҫбһ„ вҖў бһҖбһ¶бһҡбһҖбҹӮбһӣбһҳбҹ’бһўбһҷбҹүбһ¶бһ„бһ…бҹ’бһ”бһ¶бһҹбҹӢбһӣбһ¶бһҹбҹӢ бһ“бһ·бһ„бһҠбҹ„бһҷбһ”бҹ’бһҡбһҷбҹ„бһӣбҹ” вҖўбһўбһҸбһ·бһҗбһ·бһҮбһ“бһ…бҹҒбһү вҖў BAdIs вҖў BTE (бһ–бҹ’бһҡбһ№бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҺбҹҚбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһўбһ¶бһҮбһёбһңбһҖбһҳбҹ’бһҳ) вҖўбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ…бһ¶бһҖбһ…бҹҒбһү - бһҠбҹҶбһҺбһҫбһҡбһЎбһҫбһ„бһңбһ·бһүбҹ” - бһҡбһ”бһ¶бһҷбһҖбһ¶бһҡбһҺбҹҚбҹ” - SAPScripts (бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„ бһ“бһ·бһ„бһ‘бһҳбҹ’бһҡбһ„бҹӢ)бҹ” - SmartForms бһ“бһ·бһ„ SmartStyles бҹ” - бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҖбһ¶бһҡбһҠбһ№бһҖбһҮбһүбҹ’бһҮбһјбһ“ (STMS) бһҖбһ“бҹ’бһӣбҹӮбһ„бһ’бҹ’бһңбһҫбһҖбһ¶бһҡ бһ“бһ·бһ„бһҹбҹҶбһҺбһҫбһҠбһ№бһҖбһҮбһүбҹ’бһҮбһјбһ“бһҸбһ¶бһҳбһҸбһҳбҹ’бһҡбһјбһңбһҖбһ¶бһҡбҹ” - бһҹбһ»бһ–бһӣбһ—бһ¶бһ– бһҖбһ¶бһҡбһҮбҹҶбһ“бһҪбһҹ бһ“бһ·бһ„бһ…бҹ’бһ”бһ¶бһ”бҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ FI: MIRO, MIR7, FB01 бҹ”бһӣбҹ” (бһӣбһҖбҹ’бһҒбһҒбһҺбҹ’бһҢ бһ–бһ·бһ“бһ·бһҸбҹ’бһҷ бһҹбһ¶бһҡ)бҹ” - бһңбҹүбһ¶бһҡбҹ’бһҷбҹүбһ„бҹӢбһӣбһҫбһ”бһҺбҹ’бһҸбһ¶бһүбһ“бһ·бһ„бһ•бҹ’бһ‘бҹғбһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҷ; бһӢбһ·бһҸбһ·бһңбһ“бҹ’бһҸ бһ“бһ·бһ„бһҗбһ¶бһҳбһңбһ“бҹ’бһҸбҹ” - бһ•бҹ’бһҹбҹҒбһ„бһ‘бҹҖбһҸбҹ– бһ’бһ¶бһҸбһ»бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ бһҠбҹӮбһ“ бһ‘бһ·бһҠбҹ’бһӢбһ—бһ¶бһ– (бһ”бһ„бҹ’бһ бһ¶бһү бһҖбһ¶бһҡбһҗбҹӮбһ‘бһ¶бҹҶ бһҹбһ»бһ–бһӣбһ—бһ¶бһ–/бһ–бҹ’бһҡбһ№бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҺбҹҚбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһҠбҹ„бһҷбһҒбҹ’бһӣбһҪбһ“бһҜбһ„) бһҸбһ¶бһҡбһ¶бһ„ бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’ бһ”бҹ’бһҡбһ—бҹҒбһ‘бҹ”бһӣбҹ”

бһҳбһёбһ“бһ¶ 2014 - бһҳбһ·бһҗбһ»бһ“бһ¶ 2014 · 4 бһҒбҹӮ · Seville бһ“бһ·бһ„бһҸбҹҶбһ”бһ“бҹӢбһҮбһ»бҹҶбһңбһ·бһү

вңҳ бһҮбҹҶбһ“бһ¶бһү бһ“бһ·бһ„бһ…бҹҶбһҺбҹҒбҹҮбһҠбһ№бһ„бһҠбҹӮбһӣбһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһ“бҹ…бһўбҹҒбһҹбһўбҹҒбһҹ - бһ’бҹ’бһңбһҫбһҹбһңбһ“бһҖбһҳбҹ’бһҳ бһ“бһ·бһ„бһҡбҹҖбһ”бһ…бҹҶбһ‘бһјбһҡбһ‘бҹҶбһ“бһ¶бһҖбҹӢбһ‘бҹҶбһ“бһ„бһҖбҹ’бһ“бһ»бһ„бһҳбһҺбҹ’бһҢбһӣбһҹбһ»бһҒбһ—бһ¶бһ– бһ“бһ·бһ„бһҳбһҺбҹ’бһҢбһӣбһҹбһ»бһҒбһ—бһ¶бһ–бҹ” - бһҠбҹҶбһЎбһҫбһ„ бһ“бһ·бһ„бһҮбһҪбһҹбһҮбһ»бһӣбһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡ бһӮбҹ’бһҡбһҝбһ„бһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡ бһ“бһ·бһ„бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһ–бҹ’бһҡбһёбһ“бҹ” бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҹбҹ’бһҗбһ¶бһ“бһёбһҷбһҖбһ¶бһҡбһ„бһ¶бһҡбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбҹ” - бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„ бһ“бһ·бһ„бһҖбһ¶бһҡбһҠбҹҶбһЎбһҫбһ„ Citrix, Diraya, Symantec Management Console/Agent, UltraVNC, Look LAN бһ“бһ·бһ„ Mozilla Thunderbird бһҖбҹ’бһ“бһ»бһ„бһ…бҹҶбһҺбҹ„бһҳбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ•бҹ’бһҹбҹҒбһ„бһ‘бҹҖбһҸбҹ” - бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҡбһ”бҹҖбһ”бһңбһ¶бһҡбҹҲбһңбһ·бһҮбҹ’бһҮбһ¶бһҮбһёбһңбҹҲбһҗбҹӮбһ‘бһ¶бҹҶбһҹбһ»бһҒбһ—бһ¶бһ–бҹ” - бһҖбһ¶бһҡбһӮбһ¶бҹҶбһ‘бҹ’бһҡбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ’бҹ’бһңбһҫбһ…бҹҶбһҺбһ¶бһҖбһҹбҹ’бһҡбһ»бһҖбһ“бҹғбһ”бһҺбҹ’бһҸбһ¶бһү RCJA бһ–бһёбһҖбҹҶбһҺбҹӮ 2 бһ‘бҹ…бһҖбҹҶбһҺбҹӮ 3 (v2 -> v3) бҹ” бһ‘бһјбһҡбһҹбҹҗбһ–бҹ’бһ‘ IP бҹ”

бһңбһ·бһүбҹ’бһүбһ¶бһ”бһ“бһ”бһҸбҹ’бһҡ

бһўбһ¶бһҮбҹ’бһүбһ¶бһ”бһҺбҹ’бһҺ бһ“бһ·бһ„бһңбһ·бһүбҹ’бһүбһ¶бһ”бһ“бһ”бһҸбҹ’бһҡбһҠбҹӮбһӣбһ”бһ¶бһ“бһ•бҹ’бһҹбһ–бҹ’бһңбһ•бҹ’бһҹбһ¶бһҷбһ“бҹ…бһӣбһҫбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһҡбһјбһ”бһңбһ·бһҮбҹ’бһҮбһ¶бһҮбһёбһңбҹҲбһҡбһ”бһҹбҹӢбһҒбҹ’бһүбһ»бҹҶбҹ”
